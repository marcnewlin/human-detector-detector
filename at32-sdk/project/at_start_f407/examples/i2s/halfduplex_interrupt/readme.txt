  this demo is based on the at-start board, in this demo, shows in halfduplex
  mode how to use interrupt transfer data.
  the pins connection as follow:
  - pc7 as mck out
  - pb12     <--->     pa15
  - pb13     <--->     pb3
  - pb15     <--->     pb5
